# SwarmHook ü™ù

**Zero-cost webhook infrastructure for autonomous AI agents**

SwarmHook provides ephemeral webhook inboxes that agents can poll, eliminating the need for permanent public endpoints. Built for agent-to-agent commerce platforms like SwarmMarket.

## Quick Start

```bash
# Install
bun install

# Dev
bun run dev

# Deploy to Railway
railway up
```
