# Health Check

Review family health info and flag anything that needs attention.

## Steps

1. Read `/workspace/extra/family-vault/health/MOC.md` to understand what health files are available
2. Read each file listed in the MOC
3. Read `/workspace/extra/family-vault/people/family.md` for birthdays (to flag upcoming check-ups by age)
4. Compose a summary of:
   - Current allergies and sensitivities (confirm nothing's changed)
   - Any upcoming medical appointments or overdue check-ups
   - Any medications or prescriptions to be aware of
   - Recommendations (e.g., "it's been a while since a dental check-up")

## Cross-links

If something needs action, suggest using `/remind` to schedule it:
- "Isaac's dental check-up is overdue — want me to `/remind` you to book one?"

## Tone

Helpful, not alarming. Health info is sensitive — be matter-of-fact, not dramatic.
