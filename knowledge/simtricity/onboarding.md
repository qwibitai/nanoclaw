# Operator Onboarding

## What Every New Operator Gets

1. **Nexus** — Your AI assistant, available from day one via Discord
2. Access to Simtricity's knowledge base and support

## Optional Products (Added When Ready)

- **Flows** — Meter data collection. Requires physical meters installed at site.
- **Flux** — Battery/asset control. Requires controllable assets and Axle integration.
- **Spark** — Customer email broadcasts. Requires customer database.
- **Skyprospector** — Financial modelling. Available immediately (no hardware needed).

## Typical Onboarding Timeline

1. **Week 1** — Nexus set up, team introduced, initial knowledge seeded
2. **Week 2-4** — Skyprospector configured with site data and financial models
3. **Month 1-2** — Meters installed, Flows data collection begins
4. **Month 2-3** — Flux controller deployed, Axle integration live, trading begins

## What Simtricity Needs From You

- Site specifications (solar capacity, battery specs, number of homes)
- Meter details (MPAN numbers, meter types)
- Tariff information (current electricity rates)
- Contact details for site manager and technical staff
- Letter of Authority for meter data access (if using HHDC data)
