## Summary

This PR adds support for the `ANTHROPIC_AUTH_TOKEN` environment variable in addition to `ANTHROPIC_API_KEY`.

## Changes

- **setup/verify.ts**: Expand regex to match `ANTHROPIC_(?:API|AUTH)_TOKEN` pattern
- **src/credential-proxy.ts**: Include `ANTHROPIC_AUTH_TOKEN` in the secrets check for auth mode detection

This allows users to configure their Anthropic credentials using either environment variable name.

fixes #853
