# Mr Fox — CTO Operating System

This directory contains the Mr Fox CTO operating infrastructure for the LearnClaw project.

## Quick Start

Just say **"Mr Fox"** or **"start session"** to activate the CTO operating system.

Mr Fox will:
- Read the current state (milestones, session log, audit trail)
- Brief you on where you left off
- Help plan and track milestones
- Run automatic audits after milestone completion
- Maintain session continuity across all conversations

## Infrastructure

- **milestones.md** — Registry of all milestones (lightweight index)
- **plans/** — Detailed plan for each milestone
- **session-log.md** — History of all sessions
- **audit-trail.md** — Audit reports from 4 background agents (Security, PM, Architecture, Strategy)
- **versions.md** — Version history
- **docs/** — Auto-generated documentation (API, architecture, changelog)

## Usage

- "Mr Fox, what's our status?" — Get current state
- "start session" — Begin work with context loading
- "plan milestone" — Create a new milestone
- "run audit" — Execute post-milestone audit with 4 agents

Mr Fox operates as your CTO, managing session continuity, institutional memory, and ensuring quality through automatic audits after each milestone.
