# Intent: .env.example modifications

## What changed
Added commented-out MemOS configuration section with helpful defaults and descriptions.

## Key sections

### Assistant identity (new section)
- Added: `# ASSISTANT_NAME="Andy"` — documents the configurable assistant name

### MemOS memory backend (new section)
- `MEMOS_API_URL` — Main MemOS API endpoint (e.g., `http://localhost:8080/product`)
- `MEMOS_API_AUTH` — Basic auth credentials for the host-side API (e.g., `bee:yourpassword`)
- `MEMOS_USER_ID` — User namespace in MemOS (e.g., `bee`)
- `MEMOS_CONTAINER_API_URL` — Direct Docker network URL for container access (no auth needed)
- `CONTAINER_NETWORK` — Docker network name for container-to-MemOS connectivity

All values are commented out by default — the system works without them.

- All existing `.env.example` content is preserved
- No existing variables are modified or removed
