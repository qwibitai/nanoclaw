### Task A1: Routing primitive with splat support, requireAuth HOF, CSRF Origin check

**File:** `src/dashboard/router.ts`
**Operation:** CREATE
**Test file:** `src/dashboard/router.test.ts`

**Approach:**
Implements design §2 + §2a. Tiny route table with first-match dispatch, `:param` and terminal `*tail` segments, `requireAuth` HOF that verifies the cookie and populates `ctx.user` + `ctx.scopes`, and an Origin-header CSRF check on POST/PUT/DELETE. Match the project's distributed-registry convention (`registerDeliveryAction`, `registerChannelAdapter`, `registerTools`). `dispatch()` returns `Promise<Response | null>` — `null` signals "I wrote raw to ctx.rawNodeRes; do not call fromWebResponse." Cycle-3 fix M4-c3: `RequestContext` includes `rawNodeReq: http.IncomingMessage` so SSE handlers can register `req.on('close')`.

**Interface / signature:**
```ts
type Method = 'GET' | 'POST' | 'PUT' | 'DELETE';
type Handler = (req: Request, params: Record<string, string>, ctx: RequestContext) => Promise<Response | null>;
type AuthHandler = (req: Request, params: Record<string, string>, ctx: AuthedRequestContext) => Promise<Response | null>;

interface RequestContext {
  rawNodeReq: http.IncomingMessage;
  rawNodeRes?: http.ServerResponse;
}
interface GroupScope { agent_group_id: string; role: 'owner' | 'global_admin' | 'admin_of_group' | 'member'; }
interface AuthedRequestContext extends RequestContext {
  user: User;
  scopes: { role: GroupScope['role']; allowed_group_ids: string[]; no_filter: boolean };
}

export function register(method: Method, pattern: string, handler: Handler): void;
export function requireAuth(handler: AuthHandler): Handler;
export function pathMatch(pattern: string, urlPath: string): Record<string, string> | null;
export function dispatch(req: Request, nodeReq: http.IncomingMessage, nodeRes: http.ServerResponse): Promise<Response | null>;
```

ASSERT: `pathMatch('/dashboard/api/tasks/:id', '/dashboard/api/tasks/abc')` returns `{ id: 'abc' }`.
ASSERT: `pathMatch('/dashboard/static/*tail', '/dashboard/static/assets/app.js')` returns `{ tail: 'assets/app.js' }`.
ASSERT: `pathMatch('/dashboard/static/*tail', '/dashboard/static/')` returns `{ tail: '' }` (handler may 404 on empty tail).
ASSERT: `requireAuth` returns 401 `{ error: 'unauthenticated' }` when cookie is missing or invalid.
ASSERT: `requireAuth` returns 403 `{ error: 'origin_mismatch' }` for POST/PUT/DELETE when `Origin` is set and does not match `Host` (localhost/127.0.0.1/::1 origins are accepted regardless).
ASSERT: handlers writing to `ctx.rawNodeRes` directly return `null`; `dispatch` does NOT call `fromWebResponse` for `null` returns.

**Approach:**
Implements design §7 cycle-3 schema. `dashboard_tokens` stores HMAC-of-token (NOT raw bearer); `steer_idempotency` uses pending/applied status with `request_hash` for body binding and `echo_attempted` to prevent double-echo on retry. export const migration028: Migration = {
```ts
  version: 28,
  name: 'dashboard-tables',
  up: (db: Database.Database) => {
    db.exec(`
      CREATE TABLE dashboard_tokens (
        id          INTEGER PRIMARY KEY AUTOINCREMENT,
        user_id     TEXT NOT NULL REFERENCES users(id),
        token_hmac  TEXT NOT NULL UNIQUE,
        issued_at   TEXT NOT NULL,
        expires_at  TEXT NOT NULL,
        used_at     TEXT
      );
      CREATE INDEX idx_dashboard_tokens_unused
        ON dashboard_tokens(user_id, expires_at)
        WHERE used_at IS NULL;

      CREATE TABLE steer_idempotency (
        id              INTEGER PRIMARY KEY AUTOINCREMENT,
        user_id         TEXT NOT NULL REFERENCES users(id),
        idempotency_key TEXT NOT NULL,
        task_id         TEXT NOT NULL,

        reserved_at     TEXT NOT NULL,
        applied_at      TEXT,
        cached_response TEXT,
        echo_attempted  INTEGER NOT NULL DEFAULT 0,
        request_hash    TEXT NOT NULL,
        UNIQUE(user_id, idempotency_key)
      );
      CREATE INDEX idx_steer_idempotency_age
        ON steer_idempotency(reserved_at);
    `);
  },
};
```
