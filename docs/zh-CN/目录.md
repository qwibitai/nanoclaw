## 快速开始

1. 在 WhatsApp 中找到你注册的群组
2. 发送 `@触发词 你好`（例如：`@媳妇 你好`）
3. 等待 AI 回复

## 架构概述

```
WhatsApp 消息 → NanoClaw 主进程 → Apple Container (隔离环境) → Claude AI → 回复
```

- **主进程**: Node.js 应用，处理 WhatsApp 连接和消息路由
- **容器**: 每个对话在独立的 Linux 容器中运行，保证安全隔离
- **Claude**: 通过 Claude Agent SDK 调用 AI 能力
