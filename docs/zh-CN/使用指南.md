# 使用指南

## 基本对话

在注册的 WhatsApp 群组中，以触发词开头发送消息：

```
@媳妇 今天天气怎么样？
@媳妇 帮我写一首诗
@媳妇 解释一下量子计算
```

## 主频道特权

主频道（`main` 文件夹对应的群组）拥有额外权限：

- 访问整个项目目录
- 管理其他群组
- 配置定时任务
- 读写所有群组的文件

## AI 能力

NanoClaw 的 AI 助手可以：

- **回答问题**: 任何知识性问题
- **搜索网络**: 获取最新信息
- **读写文件**: 在工作区保存笔记、数据
- **执行命令**: 在沙箱环境中运行 bash 命令
- **定时任务**: 设置提醒和周期性任务
- **发送消息**: 主动向群组发送消息

## 记忆系统

AI 助手通过 `CLAUDE.md` 文件记忆重要信息：

- `groups/global/CLAUDE.md`: 全局记忆，所有群组共享
- `groups/{群组名}/CLAUDE.md`: 群组专属记忆

你可以让助手记住信息：

```
@媳妇 记住我的生日是1月1日
@媳妇 记住项目截止日期是下周五
```

## 定时任务

### 设置一次性提醒

```
@媳妇 明天早上9点提醒我开会
@媳妇 30分钟后提醒我吃药
```

### 设置周期性任务

```
@媳妇 每天早上8点告诉我今天的天气
@媳妇 每周一早上9点发送本周工作计划
```

### 查看任务

```
@媳妇 显示所有定时任务
```

### 取消任务

```
@媳妇 取消"天气提醒"任务
```

## 管理群组

在主频道中可以管理其他群组。

### 查看可用群组

```
@媳妇 显示所有可用的WhatsApp群组
```

### 注册新群组

```
@媳妇 把"家庭群"添加为注册群组
```

### 查看已注册群组

```
@媳妇 显示所有已注册的群组
```

### 移除群组

```
@媳妇 移除"测试群"
```

## WhatsApp 格式

AI 回复会自动适配 WhatsApp 格式：

- **粗体**: `*文字*`
- **斜体**: `_文字_`
- **删除线**: `~文字~`
- **代码**: `` `代码` `` 或 ` ```代码块``` `
- **列表**: 使用 `•` 符号

## 长任务处理

对于需要较长时间的任务（如搜索、文件操作），AI 会：

1. 先发送确认消息，说明正在处理
2. 执行任务
3. 发送最终结果

这样你不需要等待沉默，知道任务正在进行中。

## 对话历史

每个群组的对话历史保存在 `groups/{群组名}/conversations/` 目录下。

AI 可以查阅历史对话来获取上下文：

```
@媳妇 我们上次讨论了什么？
@媳妇 帮我找一下之前提到的那个网站链接
```
